# Class Management System

The Class Management System is a web-based application developed to simplify and digitize the management of educational institutions. The system provides a centralized platform for administrators, teachers, and students to manage academic information efficiently.

This project allows administrators to manage student and teacher records, courses, attendance, test marks, and announcements through an intuitive interface. Teachers can record attendance, update student marks, and view student information, while students can access their profiles, attendance records, test results, and important announcements.

The application is built using HTML, CSS, and JavaScript for the frontend, Python (Flask) for the backend API, and MySQL for database management. It follows a full-stack development approach and demonstrates the implementation of database connectivity, REST APIs, and CRUD (Create, Read, Update, Delete) operations.

## Key Features

* Secure Login System
* Student Management
* Teacher Management
* Course Management
* Attendance Tracking
* Test and Marks Management
* Announcement Management
* Database Integration with MySQL
* Responsive User Interface
* RESTful API Architecture
